Code for Hiroshima

テクノロジーを活用して、より良いまちづくりと、未来を考える

Code for Hiroshimaとは?

情報技術やデザインの力で、広島に住む私たちの生活をより良くすることを目指し、 テクノロジーを活用して、身のまわりの課題を解決することを目的としたグループです。 また、被爆都市として、過去を振り返り、未来の平和について考えます。

Code for Hiroshimaでは、広島に住む私たちの課題を集め、その課題を整理・分析した上で、実際に課題解決となる仕組みや方法を開発します。 私たちは、デザインをして、ソフトウェアコードを書き、課題を解決するサービスを完成させるのです。さらに、そのサービスを提供し続ける力も持ちます。

Code for Hiroshimaは市民のための組織です。行政や民間企業の影響を受けずに中立・公益の立場から物事を判断するよう心がけています。
